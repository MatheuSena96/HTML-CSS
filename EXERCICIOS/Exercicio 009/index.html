Trabalhando com Listas

Lista Ordenada

Usamos a Tag <ol>, que significa Ordened List ou Lista Ordenada
Dentro da Tag, colocaremos outra TAG chamada <li>; que significa List Icon.
Exemplo:

  1. Acordar
  2. Lavar o cabelo
  3. Limpar o cachorro
  4. Estudar
  5. Comer
  6. Brincar com bicotin
  7. Ligar para Tamira

Podemos Também fazer personalização no inicio da lista utilizando o parâmentro TYPE dentro da TAG <OL>, escolhendo entre 5 personalizações:
1 - Valor com numeros
A - Alfabeto maiusculo
a - Alfabeto minusculo
I - Numero romano maiusculo
i - Numero romano Minusculo

A Tag ficará como <ol type =""> Como ficará esse exercicio:

  1. FAzer biscoito
  2. Comprar comida
  3. Fazer coco
  4. dormir

Lista não Ordenadas

São lista que não são numeradas, somente para verificação ou compilado de itens. São listas com marcadores.
Utilizaremos a TAg <ul> de Unordered Lists.

  • Comprar Uva
  • Comprar mamão
  • Comprar peixe
  • Comprar Cuscuz
  • Comprar Amendoin

Podemos personalizar a Tag também utilizando o parametro TYPE e podendo escolher entre:

  • Disc
  • Circle
  • Square

Lista de Definição

Lista de definição é como se fosse um dicionário, onde colocaremos um termo e logo em seguida colocaremos a definição desse termo.

Utilizaremos as Tags <DL, DT e DD>.

HTML
Linguagem de marcação para criação de conteudo de um site.
Css
Linguagem de marcação para criação de Design de um site
